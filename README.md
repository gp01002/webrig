v27.0 教學更新指南：
參考模型選擇 (Step 2)：

新功能：在第二步「骨架對位」時，左側會出現「🔍 選擇參考模型」按鈕。你可以選擇顯示藍色的人類、四足動物或鳥類幻影。

用途：學生可以觀察幻影的骨架比例，將自己的點位對準幻影的關節，學習效率倍增。

無重疊白色舞台箱：

維持了你最喜歡的乾淨視覺效果。邊緣線條清晰銳利，透明度設定在 0.5，能提供極佳的空間深度參考。

動物動作測試 (Step 4)：

整合了四肢行走 (🐾 Quadruped) 與 翅膀拍動 (🦅 Fly) 動作。

技巧提示：如果做鳥類模型，建議將 Shoulder 設為「向下」權重，這能讓翅膀拍動時腋下的皮膚不會跟著胸部亂晃。

智慧分支連線 (雙擊)：

佈點時隨時雙擊現有點位變紅，就能重新定義父節點，這對處理多肢體生物（如龍的尾巴與翅膀）非常實用。
